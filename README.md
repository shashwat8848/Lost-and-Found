# lost-and-found-system
# MINI PROJECT
Submitted by: Shashwat Singh
### DESCRIPTION ###
The lost and found system is a web-based application that helps students report and find lost within thier college premises.
Students can 
- report lost items
- report found items
- search for items that one has lost

# TECHNOLOGY USED
- HTML
- CSS
- Bootstrap
- JavaScript
- Node.js
- Express.js
- MongoDB
